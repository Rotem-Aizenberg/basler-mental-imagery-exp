## For Windows 11 Users

### Option 1: Download Pre-built Executable (Recommended)
1. Ask the repository maintainer for the pre-built `LSCIExperiment` folder
2. Copy it to your Windows 11 PC
3. Install Basler Pylon SDK (for Lab Mode): https://www.baslerweb.com/en/downloads/software-downloads/
4. Run `LSCIExperiment.exe` from the folder

### Option 2: Build It Yourself
1. Install Python 3.11: https://www.python.org/downloads/
2. Clone this repository
3. Open Command Prompt in the `APP` folder
4. Run: `python build.py` or double-click `build.bat`
5. Wait 5-10 minutes for the build to complete
6. Find the executable in `APP\dist\LSCIExperiment\`
